# Predictive-Classification-Regression-for-Donor-Identification
Using various models like lineare regression, logistic regression, KNN, Random Forest; we tried to figure out whether to send our marketing mails to th eprobable donors. We also figured out the profits we will ba making in each case.
Hired as data scientists by a non-profit losing $0.55 per donor mailer. Analyzed 6,002 records with 20 features to build a full ML pipeline — EDA, feature engineering (Recency Score, Gift Effectiveness, Average Gift Size), and both classification and regression models. XGBoost achieved 85.73% accuracy with 96.25% precision; Random Forest regressor delivered R² of 57.61%. Deployed on 2,007 unseen records, flipping the outcome to an expected profit of $0.76 per mailing — targeting only high-probability donors and reducing resource waste.
